# csvreader

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple CSV reader application that allows you to drop CSV or Excel files and view their contents.

## Demo
https://code4fukui.github.io/csvreader/

## Features
- Supports both CSV and Excel (XLSX) file formats
- Provides a filter input to search through the data
- Displays the data in a clean, easy-to-read format

## Requirements
No special requirements. The application runs in a web browser.

## Usage
1. Open the [demo page](https://code4fukui.github.io/csvreader/)
2. Drag and drop a CSV or Excel file onto the page
3. The file contents will be displayed
4. Use the filter input to search through the data
