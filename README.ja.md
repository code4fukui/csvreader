# CSVリーダー

このアプリケーションはCSVファイルをブラウザ上で読み取ることができる便利なツールです。Excel形式のファイルにも対応しています。

## デモ
[デモページ](https://code4fukui.github.io/csvreader/)

## 機能
- CSVおよびExcel形式のファイルをドロップするだけで読み取り
- キーワードでファイルの内容をフィルタリング

## 使い方
ブラウザ上でファイルをドロップするだけで読み取りができます。ファイル一覧の上部にある検索ボックスにキーワードを入力することで、内容をフィルタリングできます。
