# CSVリーダー

このアプリケーションは、CSVファイルやExcelファイルの内容をブラウザ上で簡単に確認できるツールです。ファイルをドロップするだけで、データが読み込まれ、検索機能を使って表示内容を絞り込むことができます。

## デモ
https://code4fukui.github.io/csvreader/

## 機能
- CSVファイルやExcelファイルの読み込み
- 検索ボックスによるデータのフィルタリング

## 使い方
1. ブラウザ上の[デモページ](https://code4fukui.github.io/csvreader/)にアクセスします。
2. CSVファイルまたはExcelファイルをページ上にドロップします。
3. ファイルの内容が表示されます。
4. 検索ボックスにキーワードを入力して、表示内容をフィルタリングできます。
